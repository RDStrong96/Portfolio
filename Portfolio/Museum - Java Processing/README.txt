=CONTROLS=
W - Forwards
S - Backwards
A - Strafe Left
D - Strafe Right
Q - Rotate Left
E - Rotate Right
SPACE - 3rd Person

All textures are from textures.com

Tested on:
 Processing 3.4
 Windows 10
 NVIDA GeForce GTX 1070

USER May note that the outside section of the floor is not traverseable,
this is an intentional choice to avoid conflicts with the wall and the camera.

Added feature: 'Over the Shoulder third person'
Pressing the 'space' button will set the behind the user, with a slight downwards
tilt to better show the newly drawn avatar occupying the space of the 'frontal'
camera.
